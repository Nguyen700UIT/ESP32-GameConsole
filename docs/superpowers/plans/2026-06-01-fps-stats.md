# FPS Stats Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add current, minimum, maximum, and average FPS display to every game.

**Architecture:** Add a shared pure C++ `console::FpsCounter` with injected timestamps. Each game resets the counter on entry/reset, records frames after rendering, and draws compact stats in its existing TFT UI space.

**Tech Stack:** ESP32 Arduino, PlatformIO, TFT_eSPI, C++.

    auto stats = counter.stats();
    assert(counter.hasStats() == true);
    assert(stats.current == 50);
    assert(stats.minimum == 50);
    assert(stats.maximum == 50);
    assert(stats.average == 50);

    stats = counter.stats();

    stats = counter.stats();

    return 0;
}
